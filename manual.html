How to use the client

Installation

To be able to use the client, you have to do these steps:

  1. Create a Twitter account
  2. Visit the Developer page of Twitter
    1. log in
    2. go to "My Applications", click on "Create a new application", fill out the form, and confirm
    3. Click on your new application, set the access to "Read, Write and Access direct messages", and update your settings
    4. [Re]create your access token
  3. Download the login xml file and save it right beside your JAR
  4. Insert the four cryptic strings of your application created into the login.xml.
    You can insert more than one account this way.
Now you can run the JAR.
